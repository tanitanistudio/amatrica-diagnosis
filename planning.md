## 1. エグゼクティブサマリー

### 診断概要

| 項目 | 内容 |
|---|---|
| 診断名 | アマトリカ診断 — あなたの「愛し方」タイプを知る |
| 英語名 | Amatrica Type Diagnosis |
| キャッチコピー | 「愛し方」を知れば、関係が変わる。 |
| 診断軸 | 愛着スタイル × 愛の言語の組み合わせによる複合タイプ |
| 問数 | 20問 |
| 所要時間 | 約3〜5分 |
| 結果タイプ数 | 8タイプ |
| 対象ユーザー | 恋愛・パートナーシップに関心のある18〜39歳全性別 |

### KPI

| 指標 | 目標値（初月） | 目標値（3ヶ月） |
|---|---|---|
| UU（ユニークユーザー） | 5,000 | 50,000 |
| 診断完了率 | ≥ 75% | ≥ 80% |
| SNSシェア率 | ≥ 20% | ≥ 25% |
| 平均セッション時間 | ≥ 3分 | ≥ 3.5分 |
| 再診断率（同一ユーザー） | — | ≥ 15% |
| LP → 診断開始率 | ≥ 60% | ≥ 65% |

### 戦略サマリー

アマトリカ診断は、心理学の「愛着理論（Attachment Theory）」と「愛の言語（Love Languages）」を掛け合わせた独自の複合タイプ診断。既存の単軸診断（MBTI・エニアグラムなど）との差別化として、**恋愛・パートナーシップに特化した実用性**を前面に押し出す。結果ページでは「自分のタイプ」だけでなく「相性の良いタイプ」「注意すべきパターン」も提示し、シェア・再閲覧のループを生む設計とする。

## 4. 質問設計の方針

> **担当**: @agents/psychology-agent.md
> *(以下はプロダクトエージェントによる初期方針。psychology-agent による詳細設計で上書きすること)*

### 調査依頼項目

1. **測定軸の設計**
   - 愛着スタイル軸（安定型・不安型・回避型・混乱型）の測定方法
   - 愛の言語軸（肯定的な言葉・行為・贈り物・時間・スキンシップ）の測定方法
   - 両軸をどう組み合わせて8タイプに収束させるかのロジック設計

2. **質問フォーマット**
   - 強制選択式 vs. Likert尺度 vs. シナリオ選択式の比較
   - 各形式の信頼性・妥当性の根拠

3. **質問リスト（20問）**
   - バランス設計（各軸5問 × 2軸 × 2方向）
   - 社会的望ましさバイアスを低減するためのダミー質問・逆転項目

4. **倫理的配慮**
   - 自傷・DV等のセンシティブな反応を誘発しない表現ガイドライン

### 初期方針メモ

- **形式**: シナリオ選択式（2〜4択）を基本とし、具体的な場面設定で回答しやすくする
- **所要時間設計**: 1問あたり約10〜15秒を想定 → 20問で3〜5分
- **質問順序**: ウォームアップ（日常場面）→ コア（恋愛場面）→ クロスチェック（関係性の葛藤場面）
- **倫理**: 結果ページに「あくまでも参考の一つ」の免責表記を必ず含める

## 5. 結果タイプ定義

> **担当**: @agents/psychology-agent.md
> *(詳細な各タイプの記述・相性マトリクスは psychology-agent が担当)*

### タイプ体系（仮）

愛着スタイル（安定 / 不安）× 主要な愛の言語（言葉 / 行為 / 時間 / スキンシップ）の4×2から8タイプを生成。

| タイプ番号 | タイプ名（仮） | 愛着軸 | 愛の言語軸 | キャラクターイメージ |
|---|---|---|---|---|
| Type 1 | 言葉の守護者 | 安定 | 言葉 | 安心して「好き」を伝えられる。言葉で愛を育てる |
| Type 2 | 行動の設計者 | 安定 | 行為 | 愛は言葉より行動で示す信頼型 |
| Type 3 | 時間の共有者 | 安定 | 時間 | 一緒にいる時間が一番の愛情表現 |
| Type 4 | ぬくもりの探求者 | 安定 | スキンシップ | 体温で愛を確かめる直感型 |
| Type 5 | 言葉の渇望者 | 不安 | 言葉 | 褒め言葉・確認メッセージで安心できる |
| Type 6 | 証明の求道者 | 不安 | 行為 | 「してくれた」ことで愛を測る |
| Type 7 | 密着の求道者 | 不安 | 時間 | 一人の時間が増えると不安になりやすい |
| Type 8 | 感覚の依存者 | 不安 | スキンシップ | 触れ合いが少ないと愛を感じにくい |

### 各タイプページに含める要素

- タイプ名 + アイコン（イラスト）
- 一言キャッチ（15字以内）
- 特徴（200字程度）
- 恋愛パターン（強み・落とし穴）
- 相性の良いタイプ（Top 2）
- 注意が必要なタイプ（Top 1）
- 「あなたへのアドバイス」（100字）
- シェアボタン（X / LINE / リンクコピー）

### スコアリング仕様

```typescript
// 各質問に対して2軸のスコアを付与
type QuestionScore = {
  attachment: number;   // -2〜+2（負: 不安型傾向、正: 安定型傾向）
  loveLanguage: 'words' | 'acts' | 'time' | 'touch'; // 主要な愛の言語
  weight: number;       // 質問の重み（1.0 or 1.5）
}

// 集計ロジック
type DiagnosisResult = {
  attachmentScore: number;      // 合計スコア（閾値 0 で安定/不安を判定）
  loveLanguageScores: {
    words: number;
    acts: number;
    time: number;
    touch: number;
  };
  primaryType: AmatricaType;    // 最終タイプ（8種）
  secondaryType?: AmatricaType; // ボーダー時のサブタイプ
}
```
