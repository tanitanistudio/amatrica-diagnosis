## 概要

心理学の「愛着理論（Attachment Theory）」と「愛の言語（Love Languages）」を掛け合わせた独自の複合タイプ診断です。全20問・約3〜5分で完了し、8つの「愛し方」タイプのいずれかが判定されます。

バックエンド不要のフロントエンド完結実装。登録不要・無料でご利用いただけます。

---

## 診断の特徴

- **心理学的根拠** — 愛着理論 × 愛の言語という2つの学術的理論を組み合わせた設計
- **複合8タイプ判定** — 愛着スタイル（安定型／不安型）× 主要な愛の言語（4種）
- **相性マップ付き** — 相性の良いタイプ・注意すべきパターンも提示
- **途中保存対応** — localStorage による自動保存で、中断しても続きから再開可能
- **シェア機能** — X（旧Twitter）・LINE・リンクコピーに対応

## 8つのタイプ

| タイプID | タイプ名 | 愛着スタイル | 愛の言語 |
|---|---|---|---|
| SW | 言葉の守護者 | 安定型 | 言葉 |
| SA | 行動の設計者 | 安定型 | 行為 |
| ST | 時間の共有者 | 安定型 | 時間 |
| SC | ぬくもりの探求者 | 安定型 | 触れ合い |
| AW | 言葉の渇望者 | 不安型 | 言葉 |
| AA | 証明の求道者 | 不安型 | 行為 |
| AT | 密着の求道者 | 不安型 | 時間 |
| AC | 感覚の依存者 | 不安型 | 触れ合い |

各タイプは `result.html?type=XX`（例: `?type=AW`）で直接プレビューできます。

## スコアリング仕様

### 測定軸

| 軸 | 内容 |
|---|---|
| `attachment` | 愛着スタイル（高スコア = 安定型、低スコア = 不安型） |
| `words` | 言葉の愛の言語への感度 |
| `acts` | 行為の愛の言語への感度 |
| `time` | 時間の愛の言語への感度 |
| `touch` | 触れ合いの愛の言語への感度 |

### 判定ロジック

```
1. 各質問の有効スコア = reversed ? (8 - 回答値) : 回答値
2. 主軸スコア += 有効スコア × weight（1.0〜1.5）
3. 副軸スコア += 有効スコア × 0.3（secondary 指定がある場合）
4. 全軸を最大値で正規化（0〜1）
5. attachment 正規化スコア ≥ 0.5 → 安定型、< 0.5 → 不安型
6. words / acts / time / touch の中で最大スコアの軸 → 主要な愛の言語
7. 上記2つの組み合わせで8タイプを決定
```

### localStorage キー

| キー | 内容 | 有効期限 |
|---|---|---|
| `aiknow_amatrica_progress` | 途中保存データ（回答・現在問番号） | 診断完了まで |
| `aiknow_amatrica_result` | 診断完了結果（タイプ・スコア） | 90日（推奨） |
